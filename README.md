# Movie-Best
### An application to view the details of some movies and TV shows.
## is displayed
#### 1- Popular Movies.
#### 2- Popular TV Shows.
#### 3- Top Rated Movies.
#### 4- Top Rated TV Shows.
#### 5- Upcoming Movies.
#### 6- Trending TV Shows.
### Besides, some trending people are shown and Favorite Movies ,Details.
### user selects the favorite movies by clicking heart ,Then the movie or tv will be inserted in Room Database 
### user can deletes favorites by clicking Icon of Removing
## *Libraries
### Retrofit
### MVVM Architecture
### RxJava
### Room Database
### Navigation Components
### RecyclerView
### Picasso
